# Heart-Disease-Streamlit
A heart disease risk prediction app built with Python and Streamlit, using a trained Logistic Regression model. Users enter clinical health details and get instant predictions with probability scores.
##  How to Run

1. Clone the repository
   git clone https://github.com/your-username/heart-disease-prediction.git
   cd heart-disease-prediction

2. Install dependencies
   pip install -r requirements.txt

3. Run the app
   streamlit run app.py
